# Allineato - Prototipo di sistema di abbinamento terapeuta-paziente

Questa repository contiene un prototipo di sistema di abbinamento terapeuta-paziente che utilizza lo scraping web e l'intelligenza artificiale generativa per valutare e abbinare i profili dei terapeuti alle esigenze dei pazienti.

È candidata al bando "Call for Ideas" per l'anno 2026 dell'Università di Bologna.
