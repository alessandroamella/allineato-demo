# Allineato - prototipo di sistema di matching terapeuta-paziente

Prototipo di sistema di matching tra terapeuti e pazienti sviluppato in TypeScript con Bun.js. Valuta i profili dei terapeuti in base ai criteri del paziente (`evaluator/mock-profile.ts`), e fornisce un'interfaccia grafica per visualizzare i risultati. Idea candidata al bando "Call for Ideas" per l'anno 2026 dell'Università di Bologna.
